## final关键字

最终的，不可更改的

### 1. final修饰类

​	该类不允许被继承（String就是final类）

​	

### 2. final修饰变量

​	该变量为常量；

​	final修饰成员变量，必须要初始化

​	final修饰局部变量，可以不初始化



### 3. 修饰方法

​	该方法不能被重写





## 初始化块

### 分类

#### 实例化初始化块

```java
//每次实例化的时候调用一次;
//每次实例化，自动调用
//在构造器之前执行
{
    代码块
}

/*
	父类有实例化块，先执行父类的;
*/

```



#### 类初始化块

```java
//类加载时，只执行一次
static{
    
}
```



#### ==注意：==

执行顺序：父类的类初始化块-->子类的类初始化块-->父类实例化块-->父类构造器-->子类实例化块-->子类构造器
